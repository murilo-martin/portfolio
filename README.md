# Portifolio

Site pessoal de portfolio com apresentacao, links de contato e lista de projetos em um menu lateral (offcanvas).

## Recursos
- Alternancia de tema claro/escuro
- Animacoes sutis (foto e seta de chamada para projetos)
- Menu de projetos no topo com offcanvas do Bootstrap
- Links diretos para GitHub e LinkedIn

## Estrutura
- `HTML/index.html` pagina principal
- `CSS/style.css` estilos e animacoes
- `JS/script.js` logica de tema e icones
- `Images/` assets locais

## Tecnologias
- HTML5
- CSS3
- JavaScript
- Bootstrap 5

## Como executar
1. Abra `HTML/index.html` no navegador.
2. (Opcional) Use um servidor estatico para evitar restricoes de recursos locais.

## Personalizacao rapida
- Atualize o nome, cargo e links em `HTML/index.html`.
- Troque as cores e animacoes em `CSS/style.css`.
- Ajuste os icones e o tema em `JS/script.js`.
